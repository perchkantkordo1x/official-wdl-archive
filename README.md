# official-wdl-archive

本仓库用于归档和发布多个独立 HTML 页面。这些页面不限定于任何单一域名或具体网站，旨在以静态形式保存和展示各类网页内容。

## 项目简介

`official-wdl-archive` 是一个通用 HTML 页面归档仓库。你可以在这里找到经过整理的历史页面快照、备份文件以及其他独立网页资源。仓库本身不提供搜索、导航或动态服务，仅作为静态内容的存储与分发点。

## 目录说明

- `pages/`：存放归档的 HTML 页面文件，按日期或主题子目录组织。
- `assets/`：页面引用的样式、脚本、图片等静态资源。
- `index.html`：可选的首页文件，用于简单索引或说明。
- `README.md`：本文件，仓库说明。

每个归档页面通常保持原始结构，但可能经过必要的清理或压缩以减小体积。

## 页面归档说明

归档的 HTML 页面可能来自不同来源、不同时期。它们以原样保存，但可能进行了以下处理：

- 移除外部依赖（如外链脚本、字体、追踪代码）。
- 内联必要的样式与脚本，确保离线可查看。
- 添加归档时间戳或备注。

**注意**：归档页面仅代表特定时间点的快照，不反映后续更新或变更。内容可能不完整或与原始版本存在差异。

## 维护说明

- 本仓库由维护者不定期更新，新增或更新归档页面。
- 不承诺定期更新或长期保留所有内容。
- 页面内容仅供存档参考，请勿用于商业用途或违反原内容条款。
- 如需提交归档请求或报告问题，请通过 GitHub Issues 联系。

## 使用方式

你可以通过以下方式使用本仓库：

1. **直接浏览**：访问 `https://<你的GitHub用户名>.github.io/official-wdl-archive/` 查看在线版本（若已启用 GitHub Pages）。
2. **克隆仓库**：`git clone https://github.com/<你的GitHub用户名>/official-wdl-archive.git` 在本地查看。
3. **下载单个页面**：直接在 `pages/` 目录中找到需要的 HTML 文件并下载。
